This is Shika Tick Data.

This project comprises a desktop application that captures historical tick data from DERIV via the website's API. The data is then stored in a local database.

Currently, you are required to choose the currency pair, start time, and end time of the data you need. It is my plan to increase the number of features in the application based on common user needs, so I'd appreciate your suggestions.
